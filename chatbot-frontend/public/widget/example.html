MarvChat Widget

This is an example page demonstrating the MarvChat widget. Look for the chat button in the bottom-right corner!

How to Add to Your Site

Add this single line before the closing </body> tag:

<script src="https://chat.marv-dev.com/widget/marvchat-widget.js"></script>
Note: Replace the URL with your actual deployed chatbot URL.

Features

  • Floating chat button (bottom-right)
  • Click to open chat modal
  • Click outside or press Escape to close
  • Responsive - works on mobile
  • Loads chatbot in widget mode (no sidebar)

Customization

The widget uses these default settings:

  • Button color: Amber (#D97706)
  • Position: Bottom-right, 24px from edges
  • Modal size: 450px wide, 600px tall (responsive)

For custom styling, you can override the CSS variables or modify the widget script.
